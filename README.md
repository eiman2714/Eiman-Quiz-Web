 Interactive Quiz Application

An interactive quiz app built with HTML, CSS, and JavaScript. Test your knowledge across multiple categories with a timer and instant feedback.

 Features
- Multiple categories to choose from (General Knowledge, Science, History, Geography, Entertainment)
- 10-second timer per question
- Live score tracking during the quiz
- Final score breakdown (correct/incorrect answers)
- Retry option with same category
- Dynamic rendering — no page reloads
- Fully responsive design

 Built With

- HTML5
- CSS3
- JavaScript (ES6)

 How It Works

1. Select a category
2. Answer 5 questions within 10 seconds each
3. Get instant feedback on each answer
4. View your final score and breakdown
5. Retry anytime
